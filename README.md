## :notebook_with_decorative_cover: Descrição

Um simples programa que lê e mostra alguns dados sobre os jogos que foram lançados para o console super futurista direto do ano **2000** da Sony.

---

## :mag_right: Funcionalidades

1. Total de Jogos, Desenvolvedoras e Publicadoras Únicas
O programa descreve o total de empresas (publicadoras e desenvolvedoras únicas) assim como também quantos jogos foram lançados até o fim da vida do console.
Com isso, o usuário também possui a opção entre gerar ou não um gráfico em barras dessas informações.

2. Primeiros 20 Lançamentos
Ele mostra quais foram os primeiros 20 jogos lançados para o mesmo.

3. Jogos Lançados Mundialmente
Realiza uma soma de todos os lançamentos de forma global (sendo as regiões América do Norte, Europa e Japão)

4. Número de Exclusivos por Região
Por meio de uma diferença o programa obtem o número entre os jogos lançados de forma exclusiva nas regiões já mencionadas.
Novamente, assim como na primeira opção, o usuário pode ou não gerar um gráfico, só que desta vez, um gráfico de "pizza".

## :tv: Pré-Requisitos
- Python 3.10+

### Como Instalar as Dependências

Abra o terminal de sua preferência (PowerShell meu querido) e digite o comando:

```bash
pip install -r requirements.txt
```

Ele irá instalar as dependências do projeto automaticamente

Para rodar o programa, novamente, utilizando seu terminal favorito use o comando:

```bash
python jogos_ps2.py
```
